# Next.js Speech To Text OpenAI

# Install

**1. Clone Project**

```
  $ https://github.com/fsarikaya96/speech-to-text-openai-fe.git
```

**2. Run Project**

```
  $ npm install
  $ npm install axios
  $ npm install sweetalert2
  $ npm install react-spinners
  $ npm run dev

  http://localhost:3000
```
